* Pelestarian budaya dan lingkungan berbasis **kearifan lokal Terranesia**
* Mengadaptasi nilai-nilai tersebut agar bisa diterapkan di kehidupan modern
* Menggabungkan teknologi canggih: **Web GIS, AI, AR, VR**

Website ini tidak hanya informatif, tetapi juga:

Interaktif

# 🌍 FITUR UTAMA: JELAJAH (WEB GIS SYSTEM) ⭐ CORE INNOVATION

## 🎯 TUJUAN

Membuat sistem eksplorasi budaya berbasis peta interaktif (Web GIS) yang memungkinkan user melihat, memahami, dan membandingkan budaya di seluruh Indonesia.

---

## 🗺️ WEB GIS INTERACTIVE MAP

### 🔧 TEKNOLOGI:

* Mapbox GL JS / Leaflet.js
* GeoJSON dataset budaya Indonesia
* Integrasi backend (Firebase/MongoDB)

---

## 🧠 FITUR DETAIL:

### 1. 🌏 PETA INTERAKTIF INDONESIA

#### Behavior:

* User bisa:

  * Zoom in/out
  * Drag map
  * Klik wilayah (provinsi/kabupaten)

#### Data Layer:

* Layer 1: Budaya
* Layer 2: Lingkungan
* Layer 3: Kearifan Lokal

---

### 2. 📍 INTERACTIVE MARKER SYSTEM

Setiap titik di peta memiliki marker:

#### Ketika marker diklik:

Muncul **popup card modern** berisi:

* Nama budaya
* Lokasi
* Thumbnail gambar
* Deskripsi singkat
* Tombol:

  * “Pelajari Lebih Dalam”
  * “Lihat dalam AR/VR”

---

### 3. 🧬 FILTER & SEARCH SYSTEM

#### Filter:

* Jenis budaya:

  * Tradisi
  * Ritual
  * Sistem lingkungan
* Tingkat keberlanjutan
* Wilayah

#### Search:

* Search bar dengan AI suggestion:

  > “Cari budaya yang mirip Terranesia”

---

### 4. 🤖 AI RECOMMENDATION ENGINE

#### Sistem:

* AI menganalisis:

  * Preferensi user
  * Aktivitas eksplorasi

#### Output:

* Rekomendasi:

  > “Budaya lain yang punya konsep pelestarian seperti Terranesia”

---

### 5. 🔄 COMPARISON MODE (INOVASI 🔥)

User bisa:

* Pilih 2 budaya
* Bandingkan:

#### Data yang dibandingkan:

* Cara menjaga lingkungan
* Sistem hidup
* Nilai filosofi

#### UI:

* Split screen comparison

---

### 6. 🧭 STORY MODE (INTERACTIVE JOURNEY)

#### Konsep:

User mengikuti perjalanan:

> “Menjadi masyarakat Terranesia selama 1 hari”

#### Interaksi:

* Pilihan keputusan
* Dampak ke lingkungan

---

### 7. 🥽 GIS + VR INTEGRATION

#### Fitur:

* Klik lokasi Terranesia → masuk ke:

  * Virtual Tour 360°

#### Teknologi:

* WebXR
* Three.js

---

### 8. 📊 DATA VISUALIZATION

#### Grafik:

* Tingkat kelestarian budaya
* Ancaman urbanisasi

#### Format:

* Chart interaktif (hover animation)

---

### 9. 📡 REAL-TIME UPDATE SYSTEM

#### Backend:

* Admin bisa update data budaya

#### Frontend:

* User langsung lihat update
